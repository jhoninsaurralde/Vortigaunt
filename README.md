# Vortigaunt Homelab

![License](https://img.shields.io/badge/license-MIT-green)
![Version](https://img.shields.io/badge/version-v0.1.0-blue)
![Platform](https://img.shields.io/badge/platform-Linux-orange)

Sistema personal de monitoreo, registro y memoria persistente
para un nodo Linux doméstico.

> Observar. Registrar. Recordar. Persistir.

## Sobre el proyecto

Vortigaunt nace como un sistema ligero para dar identidad,
historial y memoria a un servidor doméstico de bajos recursos.

No busca reemplazar soluciones empresariales de monitoreo,
sino crear una capa simple y transparente sobre Linux.

## Características

- Monitoreo periódico del sistema
- Registro histórico de eventos
- Estado del hardware
- Verificación RAID1
- Reportes automáticos
- Sistema de hitos
- Registro de experiencias
- Backups automáticos
- Integración con systemd timers

## Arquitectura

Sistema base:

- Linux
- Bash scripting
- systemd
- mdadm RAID1

Hardware de referencia:

- Intel Celeron 847 @ 1.10 GHz
- 10GB DDR3 RAM
- Intel HD Graphics 2000
- RAID1 de discos mecánicos

## Filosofía

Vortigaunt no intenta ser complejo.

Su objetivo es que un equipo olvidado pueda mantenerse
activo, registrar su historia y continuar funcionando.
