# [v0.1.0] - 2026-07-21

## Added

- Initial public release of Vortigaunt.
- Persistent monitoring system for Linux home servers.
- Event logging system.
- Experience and milestone tracking.
- RAID1 health verification.
- Automated backup system.
- systemd service and timer integration.
- Configurable installation path.
- Installation and architecture documentation.
- MIT open source license.

## Hardware Reference

Genesis deployment running on:

- Intel Celeron 847 @ 1.10 GHz
- 10GB DDR3 RAM
- Intel HD Graphics 2000
- RAID1 mechanical storage

## Notes

Vortigaunt Genesis represents the first stable public version.

The goal is to provide a lightweight, transparent and persistent
companion for small Linux servers and reused hardware.
